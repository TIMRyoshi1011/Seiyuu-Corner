Seiyuu Corner Assignments Simulator/Tracker

This application has 2 functions, simulation of adding objects in vanilla Object-Oriented Programming [left panel] (no save after close) and saving in a database via MySQL [right panel] (with save after close)
- Click SC to run the app (no need to exceute SC Database in MySQL)
